## Carta de presentacion

Este es mi espacio en GitHub: un lugar donde convierto ideas en software, experimento, aprendo y dejo constancia de lo que voy construyendo. Me interesa crear proyectos que no solo funcionen, sino que tambien tengan criterio, buena presentacion y una ejecucion limpia.

Si has llegado hasta aqui, seguramente te interese alguna de estas tres cosas: ver como trabajo, descubrir proyectos personales con identidad propia o encontrar a alguien que cuide tanto la parte tecnica como el resultado final. Este perfil esta pensado justo para eso.
